RAILWAY + GITHUB

1) Suba esta pasta inteira para um repositório no GitHub.
2) No Railway, crie um projeto usando Deploy from GitHub Repo.
3) Adicione um Volume e monte em /data.
4) Configure as variáveis usando o arquivo .env.example.
5) Defina BASE_URL para o domínio do Railway.
6) Faça o deploy.

IMPORTANTE
- DB_PATH deve ficar em /data/loja-system.sqlite
- Em produção, troque ADMIN_PASSWORD, SESSION_SECRET e LICENSE_SECRET.
- Se quiser domínio próprio, configure depois no Railway.
