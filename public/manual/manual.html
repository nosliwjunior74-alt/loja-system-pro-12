Manual do Dashboard Multi-Loja

  1. Extraia o ZIP completo.
  2. Abra INICIAR_DASHBOARD_LOJA_VENDA.bat ou INICIAR_DASHBOARD_MULTI_LOJA.bat.
  3. Clique em Personalizar Loja para mudar nome, subtítulo e logo.
  4. Depois use os 3 aplicativos integrados normalmente.

Esta versão foi preparada para venda e personalização de outras lojas.
