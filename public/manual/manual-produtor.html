Manual do Produtor

Entre em Painel Mestre das Lojas, cadastre a loja, defina login e senha, configure a licença e use os botões Copiar link, WhatsApp ou E-mail.

Para publicar em domínio próprio, faça o deploy e depois aponte o DNS para o serviço publicado.

Passo a passo

  1. Entre com o login do produtor.
  2. Cadastre a loja e salve.
  3. Ative a loja.
  4. Use WhatsApp ou E-mail para enviar o acesso.
  5. Se precisar reenviar acesso, use o botão correspondente e crie uma nova senha.
